  Architectural patterns established:
  - `src/state.js` single mutable container ‚Äî all importers receive live bindings.
  - Callback injection (`setXCallbacks({ ... })`) for the few unavoidable circular dependencies (e.g. `cards/playerCard.js` ‚áÑ `collection/book.js`).
  - Layering rule (Layer 1 foundation ‚Üí Layer 6 boot) ‚Äî every module imports only from strictly lower layers; verified by esbuild.
  - Window-global bridge in `src/main.js` re-exposes ~95 functions to HTML inline handlers.
