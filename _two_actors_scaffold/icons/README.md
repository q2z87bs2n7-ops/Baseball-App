# Icons

PWA icon placeholders. Before shipping, generate and place these PNG files here:

- `icon-192.png` — 192×192, `purpose: any`
- `icon-512.png` — 512×512, `purpose: any`
- `icon-maskable-512.png` — 512×512, with safe-zone padding (~10% on each side), `purpose: maskable`

A monochrome SVG (`icon-mono.svg`) is optional but recommended for iOS status bar tint.

Until real icons are added, the PWA install prompt may show a generic placeholder.
The app still functions without them.
